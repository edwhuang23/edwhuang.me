<div class='intro'>




<p>If you use Jekyll as a static site generator with GitHub Pages, you benefit from more support with setting up, updating, and troubleshooting your site.</p>




</div>




<h3>


<a id="about-github-pages-and-jekyll" class="anchor" href="#about-github-pages-and-jekyll" aria-hidden="true"><span aria-hidden="true" class="octicon octicon-link"></span></a><a href="/articles/about-github-pages-and-jekyll">About GitHub Pages and Jekyll</a>


</h3>




<p>In addition to supporting regular HTML content, GitHub Pages supports <a href="https://github.com/jekyll/jekyll">Jekyll</a>, a popular static site generator.</p>




<h3>


<a id="about-jekyll-themes-on-github" class="anchor" href="#about-jekyll-themes-on-github" aria-hidden="true"><span aria-hidden="true" class="octicon octicon-link"></span></a><a href="/articles/about-jekyll-themes-on-github">About Jekyll themes on GitHub</a>


</h3>




<p>You can install a Jekyll theme to customize the design of your GitHub Pages site without having to edit or copy CSS files. Jekyll themes package the files you need to change your site design, including any files that are stored in your assets folder, like SCSS, SASS, images, and CoffeeScript.</p>




<h3>


<a id="adding-a-jekyll-theme-to-your-github-pages-site" class="anchor" href="#adding-a-jekyll-theme-to-your-github-pages-site" aria-hidden="true"><span aria-hidden="true" class="octicon octicon-link"></span></a><a href="/articles/adding-a-jekyll-theme-to-your-github-pages-site">Adding a Jekyll theme to your GitHub Pages site</a>


</h3>




<p>You can add a Jekyll theme to your GitHub Pages site by editing your site's <strong>_config.yml</strong> file.</p>




<h3>


<a id="adding-a-jekyll-theme-to-your-github-pages-site-with-the-jekyll-theme-chooser" class="anchor" href="#adding-a-jekyll-theme-to-your-github-pages-site-with-the-jekyll-theme-chooser" aria-hidden="true"><span aria-hidden="true" class="octicon octicon-link"></span></a><a href="/articles/adding-a-jekyll-theme-to-your-github-pages-site-with-the-jekyll-theme-chooser">Adding a Jekyll theme to your GitHub Pages site with the Jekyll Theme Chooser</a>


</h3>




<p>You can use the Jekyll Theme Chooser to create a GitHub Pages site, or to change the theme of your existing GitHub Pages site.</p>




<h3>


<a id="customizing-css-and-html-in-your-jekyll-theme" class="anchor" href="#customizing-css-and-html-in-your-jekyll-theme" aria-hidden="true"><span aria-hidden="true" class="octicon octicon-link"></span></a><a href="/articles/customizing-css-and-html-in-your-jekyll-theme">Customizing CSS and HTML in your Jekyll theme</a>


</h3>




<p>You can personalize your GitHub Pages site by adding custom styles to your Jekyll theme and changing your theme's layout.</p>




<h3>


<a id="setting-up-your-github-pages-site-locally-with-jekyll" class="anchor" href="#setting-up-your-github-pages-site-locally-with-jekyll" aria-hidden="true"><span aria-hidden="true" class="octicon octicon-link"></span></a><a href="/articles/setting-up-your-github-pages-site-locally-with-jekyll">Setting up your GitHub Pages site locally with Jekyll</a>


</h3>




<p>You can set up a local version of your Jekyll GitHub Pages site to test changes to your site locally. We highly recommend installing Jekyll to preview your site and help troubleshoot failed Jekyll builds.</p>




<h3>


<a id="configuring-jekyll" class="anchor" href="#configuring-jekyll" aria-hidden="true"><span aria-hidden="true" class="octicon octicon-link"></span></a><a href="/articles/configuring-jekyll">Configuring Jekyll</a>


</h3>




<p>You can configure most Jekyll settings by editing your <em>_config.yml</em> file.</p>
